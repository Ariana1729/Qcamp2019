Mission 3: Receives the message (M1) from the sender

In this mission, we want to receive the message (M1) from the sender (a string of super secret characters). This is implemented in a NEC-string encoding protocol, and a convinient python wrapper is used to facilitate the string transfer process between Alice and Bob.

Recommended reading:
1. NEC-string encoding protocol (RR_15)
2. Description of B3_recv_message.ino and B3_recv_message.py (RR_27)

Objectives: 
1. Establish a classical communication channel from Alice to Bob
2. Receives the super secret string M1
